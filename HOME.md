# Home Directory Index

`/Users/sayori` 顶层索引。这里只记录顶层文件和文件夹的用途，方便以后回查。

## 普通项

| 文件/文件夹 | 用途 |
| --- | --- |
| `Applications/` | 当前用户级应用目录，通常存放仅当前账号可见的 App 或辅助程序。 |
| `Desktop/` | 桌面文件目录。 |
| `Documents/` | 常规文档目录。 |
| `Downloads/` | 下载目录。 |
| `Library/` | macOS 用户级应用支持、缓存、偏好设置和系统数据目录。 |
| `Movies/` | 视频目录。 |
| `Music/` | 音乐目录。 |
| `Pictures/` | 图片目录。 |
| `Public/` | 共享目录。 |
| `HOME.md` | 当前 home 顶层索引说明文件。 |
| `app-config/` | 自定义应用配置目录；当前看起来主要用于 `Downie Extension`。 |
| `player.list` | 本地播放列表/课程文件清单，内容是若干 `.ev4a` 文件路径。 |
